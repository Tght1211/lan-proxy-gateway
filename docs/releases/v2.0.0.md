# LAN Proxy Gateway v2.0.0

定位: 扩展脚本架构。

## 主要内容

- 引入全局扩展脚本支持
- 兼容基于 `main(config)` 的脚本扩展方式
- 让高级规则和自定义逻辑从模板硬编码里解耦
- 为后续 `extension` 体系打基础
