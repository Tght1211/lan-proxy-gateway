# LAN Proxy Gateway v1.2.0

定位: Chains 链式代理首版。

## 主要内容

- 加入静态住宅 IP 链式代理能力
- 支持机场节点后接住宅代理出口
- 面向 Claude、ChatGPT、Cursor 等场景补强出口链路
- 保持局域网共享主链路不变
