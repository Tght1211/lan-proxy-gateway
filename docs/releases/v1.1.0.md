# LAN Proxy Gateway v1.1.0

定位: 安装与稳定性增强。

## 主要内容

- 加入跨平台一键安装脚本
- 增加国内镜像与回退下载链路
- 支持安装时自动下载 mihomo
- 补入健康检查、自愈、日志轮转和更新能力
- 增加本地配置 Web UI 与区域策略控制
